# Spring MVC Student Registration App

This is a Spring MVC-based web application that allows students to register and view their details.  
The registered student data is stored in an H2 in-memory database and displayed on a View Student page.

---

## Features

✔ Student Registration Form  
✔ Form Validation  
✔ Save Student Data into H2 Database  
✔ View All Registered Students  
✔ Working with JSP, Spring MVC, and JSTL  
✔ MVC Pattern Implementation  

---

## Tech Stack

**Backend:** Spring MVC  
**View Layer:** JSP + JSTL  
**Database:** H2 Database (In-Memory)  
**Build Tool:** Maven  
**Language:** Java

---

## How It Works

1. User opens the registration page
2. Fills details such as:
   - Name
   - Email
   - Gender
   - Course
   - Timings
3. Submits the form
4. Data is saved into H2 DB
5. User can view all registered students on the View Students page

---

## Database Details

H2 Console URL: `http://localhost:8080/h2-console`
